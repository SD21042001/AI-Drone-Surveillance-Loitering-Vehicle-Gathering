# 🚁 AI Drone Surveillance System

An AI-powered surveillance solution that performs real-time object detection, loitering analysis, and vehicle gathering detection using computer vision and graph-based analytics.

---

## 📌 Project Overview

Traditional surveillance systems can identify objects but often struggle to understand suspicious behaviour.

This project extends object detection by analysing movement patterns and spatial relationships to identify:

- Suspicious loitering behaviour
- Vehicle gathering events
- Real-time surveillance activities
- Event history logging for forensic review

---

## 🎯 Key Features

### 👤 Loitering Detection
- Real-time person detection
- Object tracking across frames
- Movement pattern analysis
- Suspicious stationary behaviour identification
- Automatic event recording

### 🚗 Vehicle Gathering Detection
- Vehicle detection and tracking
- Graph-based spatial analysis
- Group formation detection
- Automatic gathering event generation

### 📊 Event History System
- Human-readable SQL event logs
- Timestamp-based records
- Duration tracking
- Historical event review
